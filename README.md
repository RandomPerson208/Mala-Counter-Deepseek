# Mala-Counter-Deepseek - Meditation App

🕉️ Hare Krishna chanting counter with 108 beads.

**Features:**
- Premium dark luxe UI with gold accents
- Haptic feedback (vibrates on tap)
- Gentle bead click sound
- Daily goal tracker (set your own target)
- Progress bar & goal status
- Streak counter (consecutive days)
- Best chants record (all‑time high)
- Confetti celebration on completion
- Full data persistence (saves to your phone)

**How to use:** Tap the bead or press "Tap Bead". Reset anytime.  
Add to Home Screen for a native‑app experience.
